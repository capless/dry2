Dialog Component Showcase

Interactive dialog component with AJAX content loading capabilities using HTMX integration.

The dry-dialog component creates modal dialogs that can load content dynamically via AJAX. It integrates seamlessly with HTMX for server-side content fetching and provides a clean, accessible dialog interface.

Advanced Usage

JavaScript API

You can also control dialogs programmatically:

Hidden Trigger 
// Get reference to dialog component
const dialog = document.getElementById('programmatic-dialog');

// Open the dialog programmatically
document.getElementById('open-programmatic').addEventListener('click', () => {
    const dialogElement = dialog.querySelector('dialog');
    dialogElement.showModal();
});

// Close the dialog programmatically  
document.getElementById('close-programmatic').addEventListener('click', () => {
    const dialogElement = dialog.querySelector('dialog');
    dialogElement.close();
});

Troubleshooting

Common Issues

Dialog content not loading

Ensure the URL is correct and the server is returning valid HTML. Check browser network tab for errors.

Dialog not closing after form submission

Add the HX-CloseDialog: close header to your server response to automatically close the dialog.

Styling issues

Use the dialog-class attribute to apply custom CSS classes. Remember that dialogs are rendered in the top layer.
