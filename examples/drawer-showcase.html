Drawer Component Showcase

Slide-out drawer components with AJAX content loading for enhanced user interfaces.

The drawer components provide slide-out panels that can be positioned on either side of the screen. Available as dry-drawer for basic functionality and ajax-drawer for dynamic content loading.

Key Features:

  • • Left or right positioning
  • • AJAX content loading with HTMX integration
  • • Customizable styling via attributes
  • • Backdrop click and ESC key support
  • • Smooth animations and transitions
  • • Programmatic control via JavaScript API
  • • Built-in event system

API Reference

Common Attributes

Attribute Description Default
position Drawer position, either "left" or "right" "right"
drawer-class CSS classes for the drawer element "bg-white w-80"
header-class CSS classes for the header ""
backdrop-class CSS classes for the backdrop ""
button-class CSS classes for the trigger button "bg-blue-500 text-white px-4 py-2 rounded"
trigger-content Text for the trigger button "Open Drawer" / "Load Content"
header-content Text for the drawer header (alternative to slot) "Drawer"

AJAX Drawer Specific Attributes

Attribute Description Default
url URL to fetch content from ""
trigger-type HTMX trigger type "click"
content-id ID for the content container (should be unique) Auto-generated random ID

Events

Both drawer components emit the following custom events:

Event Description
drawer:opened Fired when the drawer is opened
drawer:closed Fired when the drawer is closed

Slots

Both drawer components support the following slots for content customization:

Slot Description
header Content for the drawer header
content Main content for the drawer (only used in dry-drawer)

HTMX Integration Notes

Information about how the AJAX drawer integrates with HTMX for dynamic content loading.

The AJAX drawer component integrates with HTMX for content loading. Here are some key features:

  • Content is loaded using HTMX's hx-get attribute
  • You can control when the request is triggered with the trigger-type attribute
  • The drawer automatically opens when the request starts
  • Loading and error states are handled automatically
  • You can close the drawer programmatically by returning the HX-CloseDrawer: close header in your response
// Server-side example (Node.js/Express) app.get('/api/drawer-content.html', (req, res) => { // Do some processing... // Optionally close the drawer after processing res.setHeader('HX-CloseDrawer', 'close'); // Send the response res.send('<div>This content was loaded via AJAX</div>'); });
