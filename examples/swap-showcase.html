Swap Component

A component that can swap between two Font Awesome icons or SVG content based on state.

Basic Usage

The Swap component toggles between two states when clicked.

Plus/Minus
Heart Toggle
Theme Toggle
Mute Toggle
HTML 
<swap-component
    icon-on='<i class="fas fa-minus"></i>'
    icon-off='<i class="fas fa-plus"></i>'>
</swap-component>

<swap-component 
    icon-on='<i class="fas fa-heart" style="color: red;"></i>'
    icon-off='<i class="far fa-heart"></i>'>
</swap-component>

States

The Swap component can be initialized in active state or disabled.

Default (Inactive)
Active
Disabled
HTML 
<!-- Default state (inactive) -->
<swap-component
    icon-on='<i class="fas fa-star" style="color: gold;"></i>'
    icon-off='<i class="far fa-star"></i>'>
</swap-component>

<!-- Active state -->
<swap-component active
    icon-on='<i class="fas fa-star" style="color: gold;"></i>'
    icon-off='<i class="far fa-star"></i>'>
</swap-component>

<!-- Disabled state -->
<swap-component disabled
    icon-on='<i class="fas fa-star" style="color: gold;"></i>'
    icon-off='<i class="far fa-star"></i>'>
</swap-component>

Custom Styling

The Swap component can be styled with custom classes.

HTML 
<swap-component class="bg-gray-100 rounded-full p-1 shadow-inner hover:bg-gray-200"
    icon-on='<i class="fas fa-toggle-on text-blue-500 text-xl"></i>'
    icon-off='<i class="fas fa-toggle-off text-gray-400 text-xl"></i>'>
</swap-component>

Events

The Swap component dispatches events when its state changes.

HTML 
<swap-component id="event-demo-swap"
    icon-on='<i class="fas fa-bell"></i>'
    icon-off='<i class="fas fa-bell-slash"></i>'>
</swap-component>

<script>
    document.getElementById('event-demo-swap').addEventListener('swap:change', (e) => {
        console.log('Swap state changed!', e.detail.active);
        // Handle state change
    });
</script>

Programmatic Control

The Swap component can be controlled programmatically.

HTML 
<swap-component id="controlled-swap"></swap-component>

<script>
    const swap = document.getElementById('controlled-swap');
    
    // Set to active state
    swap.active = true;
    
    // Set to inactive state
    swap.active = false;
    
    // Toggle the state
    swap.toggle();
</script>

API Reference

Properties

Property Type Default Description
icon-on string (HTML) Chevron up icon The HTML content to show when the component is in active state
icon-off string (HTML) Chevron down icon The HTML content to show when the component is in inactive state
active boolean false Whether the component is in active state
disabled boolean false Whether the component is disabled
size string 'medium' Component size: 'small', 'medium', 'large'
class string '' Custom CSS classes to apply to the component
transition string 'fade' Transition effect between states

Methods

Method Parameters Description
toggle() None Toggles between active and inactive states

Events

Event Detail Description
swap:change { active: boolean } Fired when the component state changes
