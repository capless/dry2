SuperSelect Component

A customizable select component with search functionality and multi-select support.

Basic Select

Simple single selection with search functionality.

<super-select name="basic-select" placeholder="Choose a fruit">
    <option value="apple">Apple</option>
    <option value="banana">Banana</option>
    <option value="orange" selected>Orange</option>
    <option value="grape">Grape</option>
</super-select>

Multi-Select

Multiple selection with pill display inside the dropdown.

<super-select name="multi-select" multiple placeholder="Choose categories">
    <option value="tech">Technology</option>
    <option value="design" selected>Design</option>
    <option value="business">Business</option>
    <option value="marketing" selected>Marketing</option>
</super-select>

Custom Styling

SuperSelect with custom button and dropdown styles.

    dropdown-class="absolute w-full z-10 mt-1 bg-white border border-purple-200 rounded-lg shadow-xl">

Disabled State

SuperSelect in disabled state.

<super-select name="disabled-select" disabled>
    <option value="option1">Option 1</option>
    <option value="option2" selected>Option 2 (Selected)</option>
</super-select>

JavaScript API

Programmatic control of SuperSelect components.

API Demo

API Result

API results will appear here...
// Get reference to component
const select = document.getElementById('api-select');

// Set value programmatically
select.value = ['js', 'ts'];

// Clear selection
select.clear();

// Select all (multi-select only)
select.selectAll();

// Get current value
console.log(select.value);

API Reference

SuperSelect Component

Attribute Type Default Description
name string '' Form input name
multiple boolean false Enable multi-select mode
disabled boolean false Disable the component
placeholder string 'Select an option' Button placeholder text
search-placeholder string 'Search options...' Search input placeholder
button-class string default styles Custom button CSS classes
dropdown-class string default styles Custom dropdown CSS classes
option-class string '' Additional option CSS classes
pill-class string '' Additional pill CSS classes
max-display-length number 25 Max characters before truncation
max-height string '15rem' Maximum dropdown height

Properties

Property Type Description
value string | array Get/set selected value(s)

Methods

Method Description
clear() Clear all selections
selectAll() Select all options (multi-select only)

Events

Event Description
change Fired when selection changes
option-selected Fired when an option is selected
option-deselected Fired when an option is deselected
