Select Component Showcase

Advanced select component with search, multi-select, and Alpine.js integration.

The dry-select component provides an enhanced select experience with search functionality, multi-select capabilities, and smooth animations. It integrates seamlessly with Alpine.js for reactive behavior.

Key Features:

  • " Searchable dropdown with instant filtering
  • " Single and multi-select modes
  • " Customizable styling via attributes
  • " Keyboard navigation support
  • " Form integration with hidden inputs
  • " Programmatic API for value manipulation
  • " Accessible with proper ARIA attributes
  • " Alpine.js integration for reactive data

JavaScript API

Examples showing how to interact with the select component programmatically.

Programmatic Control

Control the select component using JavaScript methods.

Use the buttons above to test the API 
const select = document.getElementById('api-select');

// Set value (single or array for multi-select)
select.value = ['item1', 'item3'];

// Get current value
const currentValue = select.value;

// Clear all selections
select.clear();

// Select all options (multi-select only)
select.selectAll();

// Listen for changes
select.addEventListener('change', (event) => {
    console.log('New value:', event.detail.value);
});

API Reference

Attributes

Attribute Description Default
multiple Enables multi-select mode false
disabled Disables the entire component false
placeholder Text shown when no option is selected "Select an option"
search-placeholder Placeholder for search input in multi-select mode "Search options..."
button-class CSS classes for the main button Default Tailwind classes
dropdown-class CSS classes for the dropdown Default Tailwind classes
max-height Maximum height of dropdown "15rem"
name Form field name for hidden inputs ""

Methods

Method Description Parameters
value (getter) Gets current selected value(s) None
value (setter) Sets selected value(s) String or Array
clear() Clears all selections None
selectAll() Selects all available options (multi-select only) None

Events

Event Description Detail
change Fired when selection changes { value: string | string[] }

Form Integration

Form Example

Select components integrate with HTML forms using hidden inputs.

Submit the form to see the values 
<form onsubmit="handleFormSubmit(event)">
    <!-- Single select creates one hidden input -->
    <dry-select name="single_field" placeholder="Choose one...">
        <option value="option1">Option 1</option>
        <option value="option2">Option 2</option>
    </dry-select>

    <!-- Multi select creates multiple hidden inputs with same name -->
    <dry-select multiple name="multi_field" placeholder="Choose multiple...">
        <option value="item1">Item 1</option>
        <option value="item2">Item 2</option>
    </dry-select>

    <button type="submit">Submit</button>
</form>
