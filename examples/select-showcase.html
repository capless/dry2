Select Component Showcase

Advanced select component with search, multi-select, and Alpine.js integration.

The dry-select component provides enhanced select functionality with search capabilities, multi-select support, and seamless Alpine.js integration for modern web applications.

Key Features:

  • • Single and multi-select modes
  • • Built-in search functionality
  • • Customizable placeholder text
  • • Pre-selected option support
  • • JavaScript API for programmatic control
  • • Alpine.js integration
  • • Accessible with keyboard navigation

Basic Select Examples

Simple select dropdowns with various options and configurations.
<dry-select name="fruit" placeholder="Select a fruit..."> <option value="apple">Apple</option> <option value="banana">Banana</option> <option value="cherry">Cherry</option> <option value="date">Date</option> <option value="elderberry">Elderberry</option> <option value="fig">Fig</option> <option value="grape">Grape</option> </dry-select>

Multi-Select

Select multiple options with built-in search and tag display.
<dry-select multiple name="languages" placeholder="Select languages..." search-placeholder="Search languages..."> <option value="javascript">JavaScript</option> <option value="python">Python</option> <option value="java">Java</option> <option value="csharp">C#</option> <option value="php">PHP</option> <option value="ruby">Ruby</option> <option value="go">Go</option> <option value="rust">Rust</option> <option value="typescript">TypeScript</option> <option value="kotlin">Kotlin</option> </dry-select>

Pre-selected Options

Select components with pre-selected values using the selected attribute.
<dry-select multiple name="colors" placeholder="Choose colors..."> <option value="red">Red</option> <option value="blue" selected>Blue</option> <option value="green" selected>Green</option> <option value="yellow">Yellow</option> <option value="purple">Purple</option> <option value="orange">Orange</option> </dry-select>

API Reference

Select Component Attributes

Attribute Type Default Description
multiple boolean false Enable multi-select mode
placeholder string 'Select...' Placeholder text when no selection
search-placeholder string 'Search...' Placeholder for search input
name string null Form field name
disabled boolean false Disable the select component

Methods

Method Description
setValue(value) Set selected value(s)
getValue() Get current selected value(s)
clear() Clear all selections
selectAll() Select all options (multi-select only)
