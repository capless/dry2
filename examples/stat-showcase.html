Stat Component

A stat component to display key metrics or statistics with visual emphasis and optional comparison indicators.

Basic Stats

Basic stat components with different data types.

<dry-stat
    value="1234"
    label="Total Users"
    type="number">
</dry-stat>

<dry-stat
    value="75.5"
    label="Completion Rate"
    type="percentage"
    percentage-value>
</dry-stat>

<dry-stat
    value="9875.42"
    label="Revenue"
    type="currency"
    currency="USD">
</dry-stat>

Stats with Trends

Stat components with trend indicators showing increase, decrease, or neutral trends.

<dry-stat
    value="1256"
    label="New Subscribers"
    trend="up"
    trend-value="+12.5%">
</dry-stat>

<dry-stat
    value="89.2"
    label="Page Load Time"
    type="number"
    decimal-places="1"
    trend="down"
    trend-value="-5.3%">
</dry-stat>

<dry-stat
    value="540"
    label="Active Sessions"
    trend="neutral"
    trend-value="±0%">
</dry-stat>

Stats with Comparisons

Stat components with comparison text to provide context against a prior period.

<dry-stat
    value="8452"
    label="Page Views"
    trend="up"
    trend-value="+24%"
    comparison="vs. Previous Week">
</dry-stat>

<dry-stat
    value="3.2"
    label="Average Session Duration"
    type="number"
    decimal-places="1"
    trend="down"
    trend-value="-1.5%"
    comparison="vs. Last Month">
</dry-stat>

API Reference

Stat Properties

Property Type Default Description
value string '0' The value to display in the stat
label string 'Statistic' Label describing the stat
trend string undefined Trend direction: 'up', 'down', or 'neutral'
trend-value string undefined Value to display next to trend indicator
comparison string undefined Comparison text (e.g., 'vs. Previous Month')
icon string undefined HTML string for an icon (SVG recommended)
type string 'number' Value type: 'number', 'percentage', or 'currency'
decimal-places number 0 for number, 2 for currency Number of decimal places to display
currency string 'USD' Currency code for currency type values
percentage-value boolean false Whether value is already in percentage format
layout string 'vertical' Layout orientation: 'vertical' or 'horizontal'
class string '' Additional CSS classes
