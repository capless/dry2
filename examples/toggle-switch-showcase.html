Interactive Demo

Try interacting with these toggles to see the events in action.

Event Log:

// Listen for toggle events
document.addEventListener('toggle:changed', (event) => {
    console.log('Toggle changed:', {
        name: event.detail.name,
        checked: event.detail.checked,
        value: event.detail.value
    });
});

API Reference

Toggle Switch Component

Attribute Type Default Description
name string '' Form input name
checked boolean false Toggle state
disabled boolean false Disable interaction
size string 'md' Size: 'sm', 'md', 'lg'
label string '' Toggle label text
value string 'true' Value when checked
active-bg string 'bg-blue-500' Background when active
inactive-bg string 'bg-gray-300' Background when inactive
switch-color string 'bg-white' Switch thumb color

Methods

Method Description
toggle() Toggle the switch state
check() Set switch to checked state
uncheck() Set switch to unchecked state
enable() Enable the switch
disable() Disable the switch

Events

Event Detail Description
toggle:changed { checked, value, name } Fired when toggle state changes
