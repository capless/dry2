JavaScript API

Control button behavior programmatically using JavaScript methods.

Demo Button
const button = document.getElementById('api-demo-btn'); // Toggle loading state button.setLoading(!button.loading); // Toggle disabled state button.setDisabled(!button.disabled); // Change text button.setText('New Text'); // Programmatically click button.click();

API Reference

Button Properties

Property Type Default Description
variant string 'primary' Visual style: 'primary', 'secondary', 'outline', 'text', 'danger', 'success', 'warning'
size string 'md' Button size: 'sm', 'md', 'lg', 'xl'
icon string '' Font Awesome icon classes
disabled boolean false Whether the button is disabled
loading boolean false Whether the button shows loading state
href string '' Render as link with this URL
target string '' Link target attribute
type string 'button' Button type: 'button', 'submit', 'reset'

Methods

Method Parameters Description
setLoading(loading) boolean Set loading state
setDisabled(disabled) boolean Set disabled state
setText(text) string Change button text
setIcon(iconClasses) string Change icon classes
click() none Programmatically click the button
