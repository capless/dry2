Breadcrumbs Component Showcase

Navigational breadcrumb component with customizable separators, icons, and styling options.

The dry-breadcrumbs component provides a clean way to show navigational hierarchy. It supports various separator styles, icons, and custom styling while maintaining accessibility.

Key Features:

  • " Multiple separator styles (chevron, slash, bullet, arrow, custom)
  • " Font Awesome icon support
  • " Customizable styling via attributes
  • " JavaScript API for dynamic control
  • " Automatic active state management
  • " Responsive design
  • " Accessible with proper ARIA attributes
  • " Method chaining support

JavaScript API

Control breadcrumbs programmatically with the built-in API methods.

Interactive Demo

Use the buttons below to test the JavaScript API methods.

Home Products
Breadcrumb Info: 
// Get breadcrumbs instance
const breadcrumbs = document.getElementById('my-breadcrumbs');

// Add new item
breadcrumbs.addItem('New Category', '/products/new');

// Set active item by index
breadcrumbs.setActive(0);

// Change separator style
breadcrumbs.separator = 'slash';

// Get breadcrumb information
console.log(breadcrumbs.itemCount);   // Number of items
console.log(breadcrumbs.activeIndex); // Index of active item
console.log(breadcrumbs.separator);   // Current separator

// Method chaining
breadcrumbs
    .addItem('Electronics', '/electronics')
    .addItem('Phones', '/electronics/phones')
    .setActive(1);

API Reference

Attributes

Attribute Component Description Default
separator dry-breadcrumbs Separator style: 'chevron', 'slash', 'bullet', 'arrow', or custom text 'chevron'
breadcrumb_class dry-breadcrumbs Additional CSS classes for the container ''
href breadcrumb-item URL for the breadcrumb link undefined
icon breadcrumb-item HTML for CSS class icons (e.g., Font Awesome) undefined
active breadcrumb-item Whether this item is active (last item auto-set) false
class breadcrumb-item Additional CSS classes for the item ''

Properties & Methods

Property/Method Type Description Parameters
separator property Get/set current separator style string
itemCount property (readonly) Number of breadcrumb items -
activeIndex property (readonly) Index of the active item -
addItem() method Add a new breadcrumb item text, href?, active?
setActive() method Set which item is active by index index
