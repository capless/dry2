API Reference

Accordion Properties

Property Type Default Description
multiple boolean false Allow multiple sections to be open simultaneously
disabled boolean false Disable all accordion interactions

Accordion Item Properties

Property Type Default Description
title string '' The header text for the accordion section
icon string (HTML) '' Custom icon HTML to display in the header
open boolean false Whether the section is initially open
disabled boolean false Disable this specific section

Methods

Method Parameters Description
openItem(itemId) string Open a specific accordion section
closeItem(itemId) string Close a specific accordion section
toggleItem(itemId) string Toggle a specific accordion section
openAll() none Open all sections (multiple mode only)
closeAll() none Close all sections

Events

Event Detail Description
accordion:change { itemId, isOpen, openItems } Fired when an accordion section is toggled
