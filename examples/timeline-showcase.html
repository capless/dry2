Timeline Component

A timeline component to display a series of events in chronological order with customizable styling for different event types.

Basic Timeline

A simple vertical timeline with basic events.

Initial project planning and team formation.

Collected and analyzed project requirements from stakeholders.

Started coding the core functionality of the application.

API Reference

Timeline Component Properties

Property Type Default Description
layout string 'vertical' Timeline orientation: 'vertical' or 'horizontal'
class string '' Additional CSS classes for the timeline container

Timeline Item Properties

Property Type Default Description
variant string 'default' Item variant: 'default', 'success', 'error', 'warning', 'info'
date string undefined Date or timestamp for the timeline item
title string undefined Title or heading for the timeline item
icon string undefined HTML string for a custom icon (SVG recommended)
layout string parent's layout Override the parent timeline's layout for this item
class string '' Additional CSS classes for the timeline item
