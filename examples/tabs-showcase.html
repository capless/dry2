API Reference

Tabs Properties

Property Type Default Description
active-tab string first tab ID of the currently active tab
orientation string 'horizontal' Layout orientation: 'horizontal' or 'vertical'
variant string 'boxed' Style variant: 'boxed', 'pills', or 'underline'
disabled boolean false Disable all tab interactions

Tab Item Properties

Property Type Default Description
title string '' The tab title text
icon string (HTML) '' Icon HTML to display in the tab
badge string '' Badge text to display in the tab
disabled boolean false Disable this specific tab

Methods

Method Parameters Description
switchTab(tabId) string Switch to a specific tab
nextTab() none Switch to the next enabled tab
previousTab() none Switch to the previous enabled tab

Events

Event Detail Description
tabs:change { activeTab, previousTab, tab } Fired when the active tab changes
