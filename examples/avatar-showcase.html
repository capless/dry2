Avatar Component

A customizable avatar component to display user profile images with fallback options.

Avatar Sizes

The Avatar component supports different sizes.

<avatar name="John Doe" size="xs"></avatar>
<avatar name="John Doe" size="sm"></avatar>
<avatar name="John Doe" size="md"></avatar>
<avatar name="John Doe" size="lg"></avatar>
<avatar name="John Doe" size="xl"></avatar>

Avatar Shapes

The Avatar component can be displayed in different shapes.

<avatar name="John Doe" shape="circle"></avatar>
<avatar name="John Doe" shape="square"></avatar>
<avatar name="John Doe" shape="rounded"></avatar>

Avatar with Images

The Avatar component can display images, with fallback to initials when images fail.

<avatar
    name="John Doe"
    src="https://i.pravatar.cc/150?img=1">
</avatar>

Avatar with Initials

Avatar initials are auto-generated from names or manually set.

<avatar name="John Doe"></avatar>
<avatar initials="AB"></avatar>

Avatar with Custom Content

The Avatar component supports slot content for badges and icons.
2
 
<avatar name="John Doe" size="lg">
    <div class="absolute bottom-0 right-0 w-3 h-3 bg-green-500 rounded-full border-2 border-white"></div>
</avatar>

API Reference

Avatar Component

Attribute Type Default Description
src string null URL of the avatar image
name string '' User name for auto-generating initials
initials string null Manually set initials
size string 'md' Size: 'xs', 'sm', 'md', 'lg', 'xl'
shape string 'circle' Shape: 'circle', 'square', 'rounded'
alt string auto-generated Alt text for accessibility

Methods

Method Description
setImage(src, alt?) Set image source and optional alt text
setName(name) Set name for initials generation
setInitials(initials) Set custom initials
setSize(size) Set component size
setShape(shape) Set component shape
