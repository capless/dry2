Avatar Component

A customizable avatar component to display user profile images with fallback options.

Avatar Sizes

<dry-avatar name="John Doe" size="xs"></dry-avatar> <dry-avatar name="John Doe" size="sm"></dry-avatar> <dry-avatar name="John Doe" size="md"></dry-avatar> <dry-avatar name="John Doe" size="lg"></dry-avatar> <dry-avatar name="John Doe" size="xl"></dry-avatar>

Avatar Shapes

<dry-avatar name="John Doe" shape="circle"></dry-avatar> <dry-avatar name="John Doe" shape="square"></dry-avatar> <dry-avatar name="John Doe" shape="rounded"></dry-avatar>

Avatar with Images

The Avatar component can display images, with fallback to initials when images fail.

<dry-avatar name="John Doe" src="https://i.pravatar.cc/150?img=1"> </dry-avatar>

Avatar with Initials

Avatar initials are auto-generated from names or manually set.

<dry-avatar name="John Doe"></dry-avatar> <dry-avatar initials="AB"></dry-avatar>

Avatar with Custom Content

The Avatar component supports slot content for badges and icons.
2
<dry-avatar name="John Doe" size="lg"> <div class="absolute bottom-0 right-0 w-3 h-3 bg-green-500 rounded-full border-2 border-white"></div> </dry-avatar>

Image Error Handling

See how the avatar gracefully handles broken image URLs.

<dry-avatar name="Broken Image" src="https://invalid-url.com/broken.jpg"></dry-avatar> <dry-avatar name="Valid Image" src="https://i.pravatar.cc/150?img=3"></dry-avatar> <dry-avatar name="No Image"></dry-avatar>

API Reference

Avatar Component

Attribute Type Default Description
src string null URL of the avatar image
name string '' User name for auto-generating initials
initials string null Manually set initials
size string 'md' Size: 'xs', 'sm', 'md', 'lg', 'xl'
shape string 'circle' Shape: 'circle', 'square', 'rounded'
alt string auto-generated Alt text for accessibility

Methods

Method Description
setImage(src, alt?) Set image source and optional alt text
setName(name) Set name for initials generation
setInitials(initials) Set custom initials
setSize(size) Set component size
setShape(shape) Set component shape
