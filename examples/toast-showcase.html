Toast Component

Lightweight notifications that slide in and out, with configurable type, position, duration, and styling.

Basic Toast

An info toast in the default bottom-right corner.

<dry-toast message="Hello, this is an info toast!" auto-show></dry-toast>

Toast Types

Use the type attribute or static methods for different styles.

// via static methods: Toast.info('This is an info toast'); Toast.success('That worked!'); Toast.warning('Be careful…'); Toast.error('Something went wrong');

Position Variants

Pass position in options to move the toast around the viewport.

// e.g.: Toast.info('Top-Left toast', { position: 'top-left' }); Toast.success('Top-Center toast', { position: 'top-center' });

Duration & Custom Styling

Override duration, container-class, or toast-class for customization.

// 10 second toast: Toast.info('Lasts 10 seconds', { duration: 10000 }); // custom CSS classes: Toast.success('Styled toast', { toastClass: 'bg-purple-100 text-purple-800 border-l-4 border-purple-400 p-4 rounded', duration: 5000 });

API Reference

Toast Component

Attribute Type Default Description
message string '' Toast message content
type string info Toast type: info, success, warning, error
position string bottom-right Toast position on screen
duration number 3000 Auto-dismiss duration in milliseconds
auto-show boolean false Show toast automatically on load

Static Methods

Method Description
Toast.info(message, options) Show info toast
Toast.success(message, options) Show success toast
Toast.warning(message, options) Show warning toast
Toast.error(message, options) Show error toast
