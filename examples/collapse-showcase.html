Collapse Component

Toggle the visibility of content with smooth animations, creating expandable sections.

Basic Collapse

This is the collapsible content that can be shown or hidden with smooth animations.

The collapse component preserves content when collapsed (doesn't remove from DOM) and supports customizable animation timing.

Perfect for creating FAQ sections, detailed descriptions, or any content that needs to be optionally visible.

 
<!-- Basic collapse with external trigger -->
<button onclick="document.getElementById('basic-collapse').toggle()">
    Click to toggle content
</button>

<dry-collapse id="basic-collapse">
    <div class="p-4">
        <p>This is the collapsible content that can be shown or hidden with smooth animations.</p>
        <p>The collapse component preserves content when collapsed and supports customizable animation timing.</p>
    </div>
</dry-collapse>

Accordion Example

The Collapse Component is a versatile UI element that allows you to toggle the visibility of content with smooth animations. It's perfect for creating expandable sections, FAQs, and accordion-style interfaces.

The component uses CSS transitions to smoothly animate the height of the content area. It measures the content height dynamically and animates between 0 and the full height.

It also supports programmatic control through JavaScript API and can be triggered by external elements.
  • • Smooth CSS transitions
  • • Customizable animation duration and easing
  • • Programmatic control via JavaScript API
  • • External trigger support
  • • Responsive to content changes
  • • Accessible with proper ARIA attributes
  • • Custom events for state changes

Pre-opened Content

This collapse component starts in an open state by using the 'open' attribute.

You can use this to show important content by default while still allowing users to collapse it if needed.

 
<!-- Pre-opened collapse with 'open' attribute -->
<button 
    class="w-full p-4 text-left font-medium text-gray-900 hover:bg-gray-50 border-b focus:outline-none focus:ring-2 focus:ring-blue-500 focus:ring-inset"
    onclick="document.getElementById('pre-opened').toggle()">
    <div class="flex justify-between items-center">
        <span>This content starts open</span>
        <svg class="w-5 h-5 transform transition-transform rotate-180" fill="none" stroke="currentColor" viewBox="0 0 24 24">
            <path stroke-linecap="round" stroke-linejoin="round" stroke-width="2" d="M19 9l-7 7-7-7"></path>
        </svg>
    </div>
</button>
<dry-collapse id="pre-opened" open>
    <div class="p-4">
        <p class="text-gray-600 mb-3">This collapse component starts in an open state by using the 'open' attribute.</p>
        <p class="text-gray-600">You can use this to show important content by default while still allowing users to collapse it if needed.</p>
    </div>
</dry-collapse>

JavaScript API

This collapse can be controlled programmatically using the JavaScript API.

Try the buttons below to open, close, or toggle the collapse state.

 
const collapse = document.getElementById('api-demo-collapse');

// Control methods
collapse.open();    // Open the collapse
collapse.close();   // Close the collapse
collapse.toggle();  // Toggle the state

// Properties
console.log(collapse.open); // Get current state
