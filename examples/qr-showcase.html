QR Code Component

A simple custom element for generating QR codes via the QRious library. Configure via attributes or JS API.

Basic QR Code

Provide a value to generate a code (default size: 200Ã—200).

<dry-qr-code value="https://www.qwigo.com"></dry-qr-code>

Size Variants

Adjust the size attribute (pixels) to control QR code dimensions.

<dry-qr-code value="small" size="100"></dry-qr-code> <dry-qr-code value="medium" size="200"></dry-qr-code> <dry-qr-code value="large" size="300"></dry-qr-code>

Custom Colors

Use foreground and background attributes for color customization.

<dry-qr-code value="Red FG" foreground="#e11d48"></dry-qr-code> <dry-qr-code value="Yellow BG" background="#fde047"></dry-qr-code> <dry-qr-code value="Custom" foreground="#10b981" background="#1f2937"></dry-qr-code>

Error Correction Levels

Set error-correction to L, M (default), Q or H for different error correction levels.

<dry-qr-code value="L-level" error-correction="L"></dry-qr-code> <dry-qr-code value="M-level" error-correction="M"></dry-qr-code> <dry-qr-code value="Q-level" error-correction="Q"></dry-qr-code> <dry-qr-code value="H-level" error-correction="H"></dry-qr-code>

API Reference

QR Code Component

Attribute Type Default Description
value string '' Data/content for QR encoding
size number 200 Width & height in pixels
foreground string #000000 QR code color
background string #ffffff Canvas background color
error-correction string M One of L, M, Q, H
