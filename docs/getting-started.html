Getting Started

Learn how to install and use DRY2 components in your projects in just a few minutes.

Prerequisites

Browser Support: DRY2 components work in all modern browsers that support Custom Elements, including Chrome 54+, Firefox 63+, Safari 10.1+, and Edge 79+.

  • Modern Browser: Support for Custom Elements and ES6 features
  • Tailwind CSS: Required for styling (can be included via CDN)
  • Alpine.js: Optional but recommended for enhanced components

Installation

1

Include Dependencies

Add Tailwind CSS and Alpine.js to your HTML head section:

<!-- Include Tailwind CSS --> <script src="https://cdn.tailwindcss.com"></script> <!-- Include Alpine.js (optional but recommended) --> <script src="https://unpkg.com/alpinejs@3.x.x/dist/cdn.min.js" defer></script> <!-- Include Font Awesome for icons (optional) --> <link href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.4.0/css/all.min.css" rel="stylesheet">
2

Install DRY2 Components

Install via npm or download directly:

# Install via npm npm install dry2-web-components # Or download the complete bundle directly curl -O https://unpkg.com/dry2-web-components/dist/dry2.js

The npm package includes the complete bundled file in the dist/ directory.

3

Include the Bundle

Include the single bundle file to get all components:

<!-- Option 1: From CDN --> <script src="https://unpkg.com/dry2-web-components/dist/dry2.js"></script> <!-- Option 2: Local file --> <script src="path/to/dist/dry2.js"></script> <!-- Option 3: Individual components (if needed) --> <script src="path/to/dist/components/button.js"></script>

Quick Start Example

Here's a complete HTML file to get you started:

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>DRY2 Components Example</title> <!-- Dependencies --> <script src="https://cdn.tailwindcss.com"></script> <script src="https://unpkg.com/alpinejs@3.x.x/dist/cdn.min.js" defer></script> <link href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.4.0/css/all.min.css" rel="stylesheet"> <!-- DRY2 Components Bundle (All components in one file) --> <script src="https://unpkg.com/dry2-web-components/dist/dry2.js"></script> </head> <body class="bg-gray-100 p-8"> <div class="max-w-2xl mx-auto space-y-8"> <h1 class="text-3xl font-bold">DRY2 Components Demo</h1> <!-- Button Examples --> <div> <h2 class="text-xl font-semibold mb-4">Buttons</h2> <div class="space-x-4"> <dry-button variant="primary" size="md">Primary</dry-button> <dry-button variant="secondary" size="md">Secondary</dry-button> <dry-button variant="outline" size="md">Outline</dry-button> </div> </div> <!-- Avatar Examples --> <div> <h2 class="text-xl font-semibold mb-4">Avatars</h2> <div class="flex space-x-4"> <avatar-component name="John Doe" size="sm"></avatar-component> <avatar-component name="Jane Smith" size="md"></avatar-component> <avatar-component name="Alex Johnson" size="lg"></avatar-component> </div> </div> <!-- Toast Example --> <div> <h2 class="text-xl font-semibold mb-4">Toast Notifications</h2> <dry-button variant="primary" onclick="showToast()">Show Toast</dry-button> </div> </div> <script> function showToast() { // Create and show a toast notification const toast = document.createElement('toast-component'); toast.setAttribute('type', 'success'); toast.setAttribute('auto-show', 'true'); toast.textContent = 'Hello from DRY2 Components!'; document.body.appendChild(toast); } </script> </body> </html>

Tip: Save this as an HTML file and open it in your browser to see the components in action!

Common Usage Patterns

Framework Integration

DRY2 components work seamlessly with popular frameworks:

React

// Import in your React component import 'dry2-web-components/dist/dry2.js'; function MyComponent() { return ( <dry-button variant="primary" onClick={handleClick} > Click me! </dry-button> ); }

Vue.js

<!-- In your Vue template --> <template> <dry-button variant="primary" @click="handleClick" > Click me! </dry-button> </template> <script> import 'dry2-web-components/dist/dry2.js'; </script>

Customization with Tailwind CSS

Use Tailwind classes to customize component appearance:

<!-- Custom styling with Tailwind classes --> <dry-button variant="primary" class="shadow-lg hover:shadow-xl transition-shadow" > Enhanced Button </dry-button> <avatar-component name="John Doe" class="ring-4 ring-blue-500 ring-opacity-50" ></avatar-component>

Event Handling

Components emit custom events that you can listen for:

<script> // Listen for component events document.addEventListener('toast:close', (event) => { console.log('Toast closed:', event.detail); }); document.addEventListener('accordion:change', (event) => { console.log('Accordion state changed:', event.detail); }); // Programmatic control const button = document.querySelector('dry-button'); button.addEventListener('click', () => { console.log('Button clicked!'); }); </script>
