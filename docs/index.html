DRY2 Web Components

A comprehensive library of 21 modern, accessible web components built with vanilla JavaScript, Alpine.js, and Tailwind CSS.

Framework Agnostic: These components work with any modern framework or vanilla JavaScript. No dependencies required beyond the components themselves.

Key Features

  • 21 Production-Ready Components - From basic buttons to advanced carousels
  • Full Accessibility - ARIA attributes and keyboard navigation built-in
  • Tailwind CSS Integration - Consistent design system with utility classes
  • Alpine.js Enhanced - Reactive state management where needed
  • Zero Framework Dependencies - Works anywhere web components are supported

Architecture

Web Components

Built on native web standards for maximum compatibility

Alpine.js Integration

Enhanced reactivity and animations for complex components

Tailwind CSS

Utility-first styling with consistent design tokens

Installation

CDN Installation

<!-- Include dependencies --> <script src="https://cdn.tailwindcss.com"></script> <script src="https://unpkg.com/alpinejs@3.x.x/dist/cdn.min.js" defer></script> <!-- Include DRY2 components bundle (all 21 components) --> <script src="https://unpkg.com/dry2-web-components/dist/dry2.js"></script>

NPM Installation

# Install the package npm install dry2-web-components # Import in your JavaScript import 'dry2-web-components/dist/dry2.js';

Basic Usage

Quick Example

<!-- Use any component - they're all available! --> <dry-button variant="primary" size="lg"> <i class="fas fa-rocket"></i> Get Started </dry-button> <avatar-component name="John Doe" size="md"></avatar-component> <carousel-component slides='[...]' autoplay></carousel-component>

Button Component

Tag: dry-button | View Demo →

A versatile button component with multiple variants, sizes, loading states, and icon support.

Basic Usage

<dry-button variant="primary" size="md">Click me!</dry-button> <dry-button variant="secondary" disabled>Disabled</dry-button> <dry-button variant="outline" loading>Loading...</dry-button> <dry-button href="/link" target="_blank">Link Button</dry-button>

Attributes

Attribute Type Default Options Description
variant string primary primary, secondary, outline, text, danger, success, warning Button visual style
size string md sm, md, lg, xl Button size
disabled boolean false true, false Whether button is disabled
loading boolean false true, false Show loading spinner
href string - Any URL Convert to anchor tag

Accordion Component

Attribute Type Default Description

Methods

Method Parameters Description
openItem(index) number Open specific accordion item
closeItem(index) number Close specific accordion item
toggleItem(index) number Toggle specific accordion item
openAll() - Open all accordion items
closeAll() - Close all accordion items

Toast Component

Tag: toast-component | View Demo →

Notification toasts with multiple types, positioning, auto-hide timers, and global convenience methods.

Basic Usage

<!-- Using HTML element --> <toast-component type="success" auto-show duration="3000"> Operation completed successfully! </toast-component> <!-- Using static methods (recommended) --> <script> Toast.success('Success message!'); Toast.error('Error occurred!', { duration: 5000 }); Toast.warning('Warning message!'); Toast.info('Information message!'); </script>

Attributes

Attribute Type Default Options Description
type string info info, success, warning, error Toast type and color scheme
position string top-right top-left, top-right, bottom-left, bottom-right Toast position on screen
duration number 4000 Milliseconds Auto-hide duration (0 for no auto-hide)
auto-show boolean false true, false Show toast automatically on creation

Static Methods

Method Parameters Description
Toast.success(message, options) string, object Show success toast
Toast.error(message, options) string, object Show error toast
Toast.warning(message, options) string, object Show warning toast
Toast.info(message, options) string, object Show info toast
